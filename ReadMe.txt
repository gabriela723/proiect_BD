Cum se ruleaza:
1. Se lanseaza Intellij
2. Se alege File -> Import -> General -> Existing Projects into Workspace -> Browse (Select root directory) calea catre folder
3. Se alege proiectl -> src -> main -> java -> app -> Main.java
4. Se inlocuieste in fisierul hibernate.cfg.xml numele si parola la baza de date din MySQL
5. Se asigura legatura cu baza de date si se ruleaza clientul
